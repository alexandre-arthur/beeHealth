The project

beeHealth is project to get some information about the health of bee analyse the sound

To succes we use some analog mic connected to an esp32. Than we send the data to a flask server

The server will analyse the data thank to machine learning

We use FFT to analyse the sound in a first time but some other tools could be use to get others informations about the soud

The result will be sword in a database and display on a website
