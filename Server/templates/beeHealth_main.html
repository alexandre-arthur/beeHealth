The project

BeeHealth is a project to gather information about the health of bees through sound analysis.

To achieve this, we use analog microphones connected to an esp32. The data is then sent to a Flask server for analysis.

The server utilizes machine learning techniques, including the Fast Fourier Transform (FFT), to analyze the sound data.

Additionally, we may employ other tools to extract further insights from the sound.

The results will be stored in a database and displayed on a dedicated website.
